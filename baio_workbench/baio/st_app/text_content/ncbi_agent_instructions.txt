Submit your question in the textbox to query NCBI and the UCS Genome browser.

You have access to the following NCBI services: BLASTp, BLASTn, and EUTILS 

For each successful query a text summary will be given and you can also find the raw data in a generated file.

All files can be previewed except JSON files.
